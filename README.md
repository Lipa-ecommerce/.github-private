## What is LIPA?

LIPA helps merchants run a real business on WhatsApp — without drowning in messages, manual payment checks, or missed sales.

Customers chat. Orders get structured. Payments get confirmed. You stay in control from one place.

> **WhatsApp is the storefront. LIPA is what runs behind it.**
